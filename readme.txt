HiiTV  - WordPress plugin

== Description ==

This plugin allows you to manage your own TV show right from WordPress.

== Installation ==

Installing "HiiTV" can be done by using the following steps:
1. Download the plugin after purchase
2. Upload the ZIP file (hiitv.zip) through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
